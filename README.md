A simple and user-friendly web application to download high-quality audio from YouTube videos and play your favorite audio files.

- Move downloaded files to a specified directory.
- Download audio files for offline use.

## Requirements

To run this application, you need to have Python installed on your machine. You can download Python from [python.org](https://www.python.org/downloads/).
